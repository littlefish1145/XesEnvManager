# XesEnvManager
使用Golang编写，通过覆写Xes自带Python实现使用本地Python
# 如何使用？
1.下载发布版本    
2.找到xes的安装文件夹        
3.将下载的exe重命名python.exe替换xes的python(请先备份xes python作为冗余)
4.在当前xes安装目录打开终端，输入      
```powershell
./python.exe -l 
```
生成配置文件
5.修改配置文件指定python          
Enjoy!
# WSL Support
1.下载WSL服务器
2.在wsl终端启动服务器
3.在xes安装目录打开终端，输入      
```powershell
./python.exe -l 
```
更新配置文件
4.修改配置文件指定Python
